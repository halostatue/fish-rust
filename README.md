# halostatue/fish-rust

A quick plugin for [fish shell][shell] to ensure that `$HOME/.cargo/bin` is
added to `$PATH`.

[![Version][version]][tag]

## Installation

Install with [Fisher][Fisher]:

```fish
fisher install halostatue/fish-rust@v2
```

### System Requirements

- [fish][fish] 3.4+
